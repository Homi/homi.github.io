# Changelog

## 2026-04-04

- แก้บั๊กสำคัญใน `gui.js` จากประเด็นที่บันทึกไว้เดิม เช่น ป้องกัน `StartSearch()` ล้มเมื่อไม่มีตาเดิน และปรับ logic ของ honour rule ให้ถูกต้องขึ้น
- ปรับระบบแสดงผลหมากและการเลือกช่องบนกระดานให้เสถียรขึ้นด้วย `data-sq` ลดปัญหาคลิกเพี้ยนและการย้ายหมากผิดตำแหน่ง
- แก้ flow ของ `Flip Board`, `Set Board`, `Take Back`, `Auto Move` และการไฮไลต์ตาเดินล่าสุดให้ทำงานสอดคล้องกัน
- ปรับปรุง engine บางส่วนใน `search.js` และ `evaluate.js` ให้การประเมินตำแหน่งและการเรียงลำดับตาเดินถูกต้องขึ้น
- ปรับ layout และ responsive ของหน้า `bj-makruk` ให้ระยะขอบกระดาน, side panel และการแสดงผลบนมือถือดีขึ้น
- ลบฟังก์ชัน `TCViewer` ที่เลิกใช้งาน และปรับข้อความหน้าเว็บเป็นภาษาไทยให้อ่านง่ายขึ้น
- จัดตำแหน่ง `think4.png` ใหม่ให้แสดงเฉพาะตอนคอมพิวเตอร์กำลังคิด
