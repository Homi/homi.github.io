โปรแกรมเล่นหมากรุกไทยที่เขียนด้วย JavaScript ล้วน ๆ พร้อม AI ให้ผู้เล่นได้หมุนหมาย

**ฟีเจอร์หลัก:**
- AI ที่แข็งแกร่งพร้อมอัลกอริธึม Minimax
- กราฟิกส์ที่สวยงามด้วย HTML5 Canvas
- รองรับการเล่นแบบผู้เล่นสองคน
- อินเทอร์เฟซที่ใช้งานง่าย

**เทคโนโลยีที่ใช้:**
- JavaScript (ES6+)
- HTML5 Canvas สำหรับการวาดกระดาน
- CSS3 สำหรับสไตล์
- ไม่ใช้ไลบรารีภายนอก
